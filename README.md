# CAIPO – Cámara Ambiental Inteligente

CAIPO es un sistema de monitoreo y control basado en ESP32 que integra sensores ambientales, actuadores y una interfaz de usuario para supervisar y regular condiciones internas.

El sistema incluye:
- Sensores ambientales (temperatura, humedad, luz)
- Servomotores para apertura/cierre
- LEDs de estado
- Pantalla LCD para visualización
- Módulo de control remoto con joystick y botones

El objetivo del prototipo es validar el control de actuadores, la lectura de entradas y la visualización de datos en tiempo real mediante firmware en ESP32.
